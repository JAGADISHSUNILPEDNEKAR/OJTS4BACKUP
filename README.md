# Origin Project Documentation

Welcome to the Origin project documentation repository. Here you will find an extensive collection of design, architecture, flow, database, product, and API related documentation, diagrams, wireframes, and references.

## Repository Structure

Our documentation is organized categorically to simplify discovery and maintenance.

- **`docs/`**
  - **`api/`**: API definitions and contract diagrams (`API_Contacts.md`, `API_Contracts_Diagram.md`).
  - **`architecture/`**: High-level system design and deployment planning (`SystemDesign.md`, `Deployment_Plan.md`).
  - **`database/`**: Entity Relationship Diagrams (ERD) and schemas (`ERD.md`, `Database_Schema_Diagram.md`, `origin_erd.mermaid`).
  - **`flows/`**: Consumer and data flow diagrams (`Consumer_Flow.md`, `CF.md`, `DataFlowDiagram.md`).
  - **`product/`**: Product Requirements Document (PRD) (`Origin - Product Requirements Document (PRD).pdf`).
